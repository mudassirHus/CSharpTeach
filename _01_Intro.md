## Difference between Visual Studio and VS Code

* **Visual Studio** is a full-fledged **IDE (Integrated Development Environment)**.
  It provides everything needed to **write, debug, test, and develop applications** in one place.

* **VS Code** is a **lightweight code editor**.
  It is fast, extensible, and supports many languages through extensions, but it is **not a full IDE by default**.

* **IDE (Integrated Development Environment)** ‚Äî a complete environment that supports all stages of application development:
  writing code, building, debugging, testing, and deployment.

## What is a Framework?

A **framework** is a set of **rules, structures, tools, and runtime components** that provides a foundation for building and running applications.

---

## .NET Framework

The **.NET Framework** is a framework created by **Microsoft**.
It provides the structure, runtime engine, and libraries needed to run applications.

Using the .NET Framework, we can develop:

* Web applications

* Console applications

* Desktop/Windows applications

* Games

* First released in **2002 (version 1.0)**

* Latest version: **4.8.1 (2022)**

Web applications built on the .NET Framework run on **IIS (Internet Information Services)**, which is Microsoft‚Äôs web server.

## How .NET Executes Code

```
Application Code
------------------------------
CLR (Common Language Runtime)
```

Flow:

```
Application Code ‚Üí IL (Intermediate Language) ‚Üí CLR executes IL
```

Languages using CLR include:

* C#.NET
* F#.NET
* VB.NET
* (and other .NET-compatible languages)
