# Day 4: Loops in C#

Loops are used to **repeat a block of code multiple times**.  
They help us avoid writing the same code again and again.

In simple words:  
Repeat something **until a condition becomes false**

---

## 1. What is a Loop?

A **loop** is a programming construct that executes a block of code **repeatedly** based on a condition.

> The condition always returns **true** or **false**  
> The loop runs while the condition is **true** and stops when it becomes **false**

## 3. for Loop

### Definition:
The `for` loop is used when you **know in advance** how many times the loop should run.

---

### Syntax:
```csharp
for (initialization; condition; increment/decrement)
{
    // code to repeat
}
Flow of Execution:
Initialization runs once

Condition is checked

If condition is true, loop body executes

Increment/Decrement updates the value

Condition is checked again

When condition becomes false, loop stops

Example:
csharp
Copy code
for (int i = 1; i <= 3; i++)
{
    Console.WriteLine(i);
}
4. while Loop
Definition:
The while loop is used when the number of iterations is not fixed and depends only on a condition.

Syntax:
csharp
Copy code
while (condition)
{
    // code to repeat
}
Flow of Execution:
Condition is checked before entering the loop

If condition is true, loop body executes

Value is updated inside the loop

Condition is checked again

Loop stops when condition becomes false

Example:
csharp
Copy code

while (i <= 3)
{
    Console.WriteLine(i);
    i++;
}
5. do-while Loop
Definition:
The do-while loop is used when the loop body must execute at least once, even if the condition is false.

Syntax:
csharp

Flow of Execution:
Loop body executes first

Condition is checked after execution

If condition is true, loop repeats

If condition is false, loop stops
