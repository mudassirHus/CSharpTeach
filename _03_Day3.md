# Day 3: Conditional Statements in C#

Conditional statements are used to make **decisions** in a program.  
They allow the program to execute different code based on different conditions.

In simple words:  
If something is **true** → do this  
If something is **false** → do something else

---

## 1. What is a Conditional Statement?

A **conditional statement** is a statement that allows the program to choose between different paths based on a condition.

> Condition always returns **true** or **false**.

---

## 2. if Statement

### Definition:
The `if` statement is used to execute a block of code **only when a condition is true**.

### Syntax:
```csharp
if(condition)
{
    // code to execute if condition is true
}
```

### Example:
```csharp
int age = 20;

if (age >= 18)
{
    Console.WriteLine("Eligible to vote");
}
```

---

## 3. if-else Statement

### Definition:
The `if-else` statement is used when you want to execute **one block if condition is true** and **another block if condition is false**.

### Syntax:
```csharp
if(condition)
{
    // true block
}
else
{
    // false block
}
```

### Example:
```csharp
int age = 15;

if (age >= 18)
{
    Console.WriteLine("Can vote");
}
else
{
    Console.WriteLine("Cannot vote");
}
```

---

## 4. if-else-if Ladder

### Definition:
The `if-else-if` ladder is used when there are **multiple conditions** to check.

### Syntax:
```csharp
if(condition1)
{
    // code
}
else if(condition2)
{
    // code
}
else if(condition3)
{
    // code
}
else
{
    // code
}
```

### Example:
```csharp
int marks = 72;

if (marks >= 90)
{
    Console.WriteLine("Grade A");
}
else if (marks >= 75 && marks < 90)
{
    Console.WriteLine("Grade B");
}
else if (marks >= 60 && marks < 75)
{
    Console.WriteLine("Grade C");
}
else
{
    Console.WriteLine("Fail");
}
```

**Important Rule:**  
Conditions are checked **top to bottom**.  
First true condition is executed, remaining are skipped.

---

## 5. Nested if

### Definition:
A **nested if** is an if statement inside another if statement.

Used when one condition depends on another condition.

### Syntax:
```csharp
if(condition1)
{
    if(condition2)
    {
        // code
    }
}
```

### Example:
```csharp
int age = 25;
bool hasLicense = true;

if (age >= 18)
{
    if (hasLicense)
    {
        Console.WriteLine("You can drive");
    }
    else
    {
        Console.WriteLine("Get license first");
    }
}
else
{
    Console.WriteLine("Underage");
}
```

---

## 6. Ternary Operator (?:)

### Definition:
The ternary operator is a **short form of if-else**.

### Syntax:
```csharp
condition ? value_if_true : value_if_false;
```

### Example:
```csharp
int age = 20;

string result = (age >= 18) ? "Adult" : "Minor";
Console.WriteLine(result);
```

---

## 7. Nested Ternary Operator (?:)

### Definition:
A nested ternary operator is a ternary operator written inside another ternary operator.
It is used when multiple conditions need to be checked in a single statement.

### Syntax:
```csharp
condition1 ? value_if_true
           : condition2 ? value_if_true
           : condition3 ? value_if_true
           : value_if_false;
```

### Example:
```csharp
int marks = 78;

string result = (marks >= 90) ? "A Grade"
               : (marks >= 75) ? "B Grade"
               : (marks >= 50) ? "C Grade"
               : "Fail";

Console.WriteLine(result);

```

---

## 8. switch Statement

### Definition:
The `switch` statement is used when you want to compare **one variable with multiple fixed values**.

### Syntax:
```csharp
switch(variable)
{
    case value1:
        // code
        break;

    case value2:
        // code
        break;

    default:
        // code
        break;
}
```

### Example:
```csharp
int day = 3;

switch (day)
{
    case 1:
        Console.WriteLine("Monday");
        break;

    case 2:
        Console.WriteLine("Tuesday");
        break;

    case 3:
        Console.WriteLine("Wednesday");
        break;

    default:
        Console.WriteLine("Invalid day");
        break;
}
```

**Note:**  
`break` is used to stop execution and exit the switch block.

---

## 9. Relational Operators

Used to compare values.

| Operator | Meaning |
|---------|---------|
| == | equal to |
| != | not equal to |
| >  | greater than |
| <  | less than |
| >= | greater or equal |
| <= | less or equal |

---

## 10. Logical Operators

Used to combine multiple conditions.

| Operator | Meaning |
|---------|---------|
| && | AND |
| || | OR |
| !  | NOT |

### Example:
```csharp
int age = 25;
bool hasID = true;

if (age >= 18 && hasID)
{
    Console.WriteLine("Entry allowed");
}
```

---

## 11. Common Beginner Mistakes

- Using `=` instead of `==`
- Forgetting `break` in switch
- Wrong order in else-if ladder
- Writing too much logic in ternary operator
- Missing curly braces `{ }`

---

## Key Takeaways

- Conditional statements help programs **make decisions**
- `if` → single condition  
- `if-else` → two paths  
- `else-if` → multiple paths  
- `nested if` → condition inside condition  
- `switch` → fixed value comparison  
- `ternary` → shortcut for if-else  
- Order of conditions **always matters**
